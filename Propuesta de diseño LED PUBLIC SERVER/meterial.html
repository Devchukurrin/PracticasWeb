LED PUBLIC SERVER

Sean Bienvenid@s a este apartado en donde mostramos todo los componente electronicos que implementamos en desarrollo del TABLERO LED...

Material numero 1: Se utilizo Conectores hembra y macho

Se utilizaron conectores hembra y macho como parte fundamental de nuestro Tablero LED...

Material numero 2: Bombillos LED

Se utilizaro bombillos led de color azul como parte fundamental de nuestro Tablero LED...

Material numero 3: Matriz LED MAX7219

Se utilizo una Matriz LED MAX7219 como pieza fundamental en el desarrollo del Tablero LED ya que es el encargado principal de llevar cargas a las demás matrizes LEDS para que la comunicación sea sincronizada en ambas matrizes...

Material numero 4: Arduino UNO

Se utilizo una placa arduino uno como base principal y como el cerebro para el almacenamiento de la programación...

Cátodo común vs Ánodo común

Cátodo Común

Las pantallas Cathode comunes tienen todos los cátodos LED conectados entre sí y necesitan un controlador con salidas que se vuelvan altas para encender cada segmento.

Ánodo Común

Las pantallas de ánodos comunes tienen todos los ánodos LED conectados entre sí y necesitan un controlador de pantalla con salidas que se vuelvan bajas para encender cada segmento.
